This repository contains the practical work, assignments, and projects completed for the MERN Stack Development elective course in the 6th Semester. The purpose of this repository is to demonstrate full-stack web development using the MERN Stack.

The MERN stack is a popular JavaScript-based technology stack used to build modern, scalable, and responsive web applications.


1.MongoDB – NoSQL database
2.Express.js – Backend web framework
3.React.js – Frontend JavaScript library
4.Node.js – JavaScript runtime environment
5.HTML5
6.CSS3
7.JavaScript (ES6+)
8.Git & GitHub


#Features:
1.User-friendly interface
2.RESTful API development
3.MongoDB database integration
4.CRUD (Create, Read, Update, Delete) operations
5.Responsive web design

#Installation:
1. Clone the repository
git clone https://github.com/PriyaChaudhary17-coder/mernstack.git
2.Navigate to the project directory
cd mernstack
3.Install dependencies
  i.Backend
cd backend
npm install
  ii.Frontend
cd frontend
npm install
4.Start the backend server
npm start
5.Start the frontend application
npm start
npm dev
